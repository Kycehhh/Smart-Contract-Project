# CarDealership Smart Contract

## Description

The CarDealership smart contract allows users to manage cars securely on the Ethereum blockchain. It provides functionalities for adding, updating, and removing cars, with access control to ensure only the contract owner can perform these actions.

## Getting Started

### Installing

Clone the repository:
```bash
git clone https://github.com/qeewpi/metacrafters-smart-contract-project
```
### Executing program
Compile the smart contract using your preferred development environment (e.g., Remix, Hardhat, Truffle).
Deploy the smart contract to a local blockchain (e.g., Ganache) or a test network (e.g., Ropsten).

## Example Workflow
### Adding a Car
```bash
// Example of adding a car
contract.addCar("Toyota", "Camry", 2022, 25000);
```

### Updating a Car
```bash
// Example of adding a car
contract.addCar("Toyota", "Camry", 2022, 25000);
```

### Removing a Car
```bash
// Example of removing a car
contract.removeCar(2);
```
